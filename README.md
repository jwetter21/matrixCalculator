## Statement of Inspiration
This project was inspired from the Vector_Calculator project which I completed as part of CS 161 in Spring 2022 at Western Oregon University. The project was open-ended, allowing students to choose any project that demonstrated a mastery of the Python programming concepts learned in the class. I chose to create Vector_Calculstor to compliment the material that I was learning in a different class during the same term, MTH 254 (Multivariate Calculus). The original Vector_Calculator file is included for reference.

## Statement of Goals
* Decide on a reasonable list of calculations that matrixCalculator should support.
* Complete matrixTest to test the calculations as they develop in the Matrix class.
* Implement the Matrix class to perform the desired calculations.
* Implement a main file to provide an interface for a user to interact with the matrixCalculator through.
